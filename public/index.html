API Documentation

Welcome to the MyAPI documentation. Use the endpoints below to interact with our system.

Base URL

https://api.example.com

Endpoints

GET /api/messages

Retrieve all messages sorted by timestamp.

Request

  • Method: GET
  • URL: https://api.example.com/api/messages

Response

        [
            {
            "message": "Hello, World!",
            "timestamp": "2023-01-22T10:30:00.000Z"
            },
            {
            "message": "Hi again!",
            "timestamp": "2023-01-23T12:45:00.000Z"
            }
        ]

POST /api/messages

Add a new message to the system.

Request

  • Method: POST
  • URL: https://api.example.com/api/messages
  • Body:
    •                     {
                      "message": "Hello, World!"
                    }

Response

                    {
                      "message": "Message created successfully",
                      "data": {
                        "message": "Hello, World!",
                        "timestamp": "2023-01-22T10:30:00.000Z"
                      }
                    }

Error Codes

  • 400 - Bad Request
  • 404 - Resource Not Found
  • 500 - Internal Server Error
